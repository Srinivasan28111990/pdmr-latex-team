ACM TeX to DOC Conversion Checklist

  1. Initially need to check the TeX file was in UTF-8 encoding in Notepad++. If not convert to UTF-8 in Notepad++
  2. Check the TeX command \input, \include and \bibliography. Do needfull.
  3. Run ACM_TeX2DOC_Pre-Cleanup.exe - It will pre-clean the tex file. Old TeX file will place in "Backup" folder
  4. If Unexpected Junk rectify manually.
