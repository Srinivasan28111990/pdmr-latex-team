Tool and Script Details

ACM TeX to DOC Conversion

  1. ACM_TeX2DOC_Pre-Cleanup.exe - Initially pre-clean the TeX file suitable for TeX to DOC Conversion
  2. (VBA) ACM_TeX2DOC.bas - It was VBA script for Pre-clean the DOC file.

ACM Track Changes

  1. (VBA) ACM_Diff_DOC2TeX_PreCleanup.bas - Initially pre-clean the CE DOC file suitable for auto SaveAs PDF, Tagging Small Caps, Serif, Monospace. Identify figure caption etc.
  2. ACM_Diff_Generator.exe - It will create "Diff.tex" from CE Doc file. "Reject.docx" need to open manually.
    1. ACM_Diff_Generator_LITE_Version.exe - If "Diff.tex" was not created. Use this exe. For that we need to do manually "Reject.tex" and "Accept.tex"
  3. Diff_TeX_PreCleanup.exe - It will Pre-clean the "Diff.tex"

ACM Proof Creation

  1. ACM_Proof_Pre-clean_V1.exe It will Remove the content with tag of \DIFdel and remove only tag for \DIFadd. And do more preclean-up based on proof setting and it will move from "Track_Changes" to "Working" folder

ACM Binder Creation

  1. ACM_Binder_Creation.exe It will create the Mother TeX file for ACM Book for Binder creation. copyright.tex and bookseries.tex was maximum updated in the path. Numbered Chapter child TeX file, preamble will be commented for binder creation

ACM Index Pre Cleanup

  1. ACM_Index-Pre-clean.exe It will pre-clean Index TeX file. Suitable for insert the index in TeX file.

ACM Index Count

  1. ACM_Index_Count.exe It will count the Index for First level, Second level, Third level seperately and printed. Also print total Index count

ACM Binder PDF multiple copied for Final Delivery

  1. ACM_PDF_Page_Splitter.exe It will copy the Binder PDF multiple times, renamed as per Final Delivery for XML and E-Pub Delivery.

Junk Process

  1. Junk_Clear-V1.exe It will convert all the Junk character to respective TeX command with any type of Input Encoding. .

ISE Diff package creation

  1. DIF_Create.exe It will copy the Edited.tex, Diff.tex and Diff.PDF to Specified folder and Renamed. Edited.tex content was copied to DOC on same folder with correct file name and zipped.
